cluster

  1. - function in module

Cluster takes an array of number-arrays like so: 

      smeans.cluster([
        [0, 0],
        [1, 1],
        [20, 30],
        [40, 2]
      ]);
... and groups them into "clusters" according to Euclidean-distance similarity.

Usage

cluster(elements, options);

Arguments

Param Type Details
elements Array

An array of N-element arrays where each element is a number.
options object

A set of options to pass to the clustering function
